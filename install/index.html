01

打開 Terminal

Terminal 是電腦的「打字介面」，平常我們用滑鼠點，待會我們要用打字的。打開方式：

  1. 按 ⌘ Cmd + Space 打開 Spotlight 搜尋
  2. 輸入 Terminal
  3. 按 Enter 開啟

會跳出一個白底（或黑底）的視窗，最上面寫著你的使用者名稱。看起來大概像這樣：

你的名字 — -zsh
Last login: Sat Apr 25 10:24:18 on ttys000 ~ %

PowerShell 是電腦的「打字介面」，平常我們用滑鼠點，待會我們要用打字的。打開方式：

  1. 按 ⊞ Win 打開開始選單
  2. 輸入 PowerShell
  3. 找到 Windows PowerShell → 點開

會跳出一個深色視窗。看起來大概像這樣：

Windows PowerShell
Windows PowerShell Copyright (C) Microsoft Corporation. All rights reserved. PS C:\Users\You>
⚠️
注意：搜尋結果裡可能也會出現「命令提示字元」，不要開那個。確認開到的視窗提示符開頭有 PS 才對，沒有的話關掉重開 Windows PowerShell。
02

貼上安裝指令

下面這串長得像火星文，先別怕。整個複製貼到 Terminal 裡，按 Enter 就好。

curl -fsSL https://claude.ai/install.sh | bash
irm https://claude.ai/install.ps1 | iex

按下 Enter 之後，等個 1~2 分鐘。畫面會跑很多字，那是電腦在偷偷幫你下載安裝，正常的。看到類似下面這樣的訊息，就代表裝好了：

安裝中
Downloading Claude Code... Installing to ~/.local/bin/claude Updating PATH... ✓ Claude Code installed successfully!
💡
如果安裝結束時出現 Setup notes 的提示，說 ~/.local/bin is not in your PATH，先在同一個視窗跑這行再繼續：
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc
🔁
裝完請把 Terminal 關掉，再重新打開一次。這樣電腦才會「認得」剛剛裝好的 claude 指令。沒重開的話下一步會找不到指令，會白擔心一場。
03

確認版本，看看裝好了沒

在重新打開的 Terminal 裡，輸入這一行，按 Enter：

claude --version

如果看到一行版本號（會像這樣，數字可能不太一樣）：

確認安裝
~ % claude --version 2.0.x (Claude Code) ~ %
🎉
恭喜，Claude Code 正式進駐你的電腦了。你現在比 90% 的人類都更接近會用 AI 寫程式。
04

啟動 Claude，看看它的家

輸入這一個字，按 Enter，正式打開 Claude Code：

claude

第一次啟動，會請你選一個主題顏色（深色 / 淺色 / 其他）。用方向鍵 ↑ ↓ 選你喜歡的，按 Enter 確認。

然後會跳出登入畫面，大致長這樣：

Claude Code
✦ Welcome to Claude Code Sign in to start using Claude Code. ▸ Log in with your Claude account Use API key (Anthropic Console)
✨
看到這個畫面，就代表你 100% 安裝成功了！今天到這裡為止，先按 Ctrl + C 離開（Mac 也是 Control + C，不是 Command）。

下一堂課，我們真的開始用

今天我們把工具裝好了，但還沒登入是因為老師會在下一堂發給大家「鑰匙」（API key）。下次上課把它貼進去，就可以正式跟 Claude 對話、請它幫你寫程式了。

先把 Terminal 關掉，今天功課完成 ✓

裝不起來？看這裡

「command not found: claude」或「'claude' 不是內部或外部命令」

意思是電腦還沒「認得」剛裝好的指令。

解法：把 Terminal 完全關掉，再重新打開一次，再試一次 claude --version。

還是不行的話，可能 PATH 沒設好。請老師看一下你的安裝訊息有沒有跑出特殊提示。

安裝過程一直卡住沒反應

通常是網路問題。檢查你的網路連線，然後 Ctrl+C 取消，再重跑一次安裝指令。

如果你在公司 / 學校網路下，可能有 firewall 擋住下載。換 wifi 或開手機熱點試試。

Windows 跳出「執行原則」（execution policy）錯誤

PowerShell 預設不允許執行外部 script。在 PowerShell 裡先跑這行，再重試安裝指令：

Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

會問你 yes/no，輸入 Y 按 Enter。這只會影響你的帳號（CurrentUser），不影響電腦其他使用者，安全。

我打開的視窗是黑底、提示符長 C:\>（沒 PS 開頭）

那是「命令提示字元」（CMD），不是我們要的 Windows Terminal。

解法：關掉，重新從開始選單搜尋 PowerShell，開 Windows PowerShell（不是命令提示字元）。

真的找不到 Windows Terminal 又急著裝的話，可以在 CMD 裡用這條備案指令：

curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd
